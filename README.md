# My Portfolio

A stunning, responsive portfolio website built with HTML, Tailwind CSS, and Lucide Icons.

## How to Run

### Option 1: VS Code (Recommended)
1. Open this project in VS Code.
2. Press `F5` or go to the **Run and Debug** tab.
3. Select **"Launch index.html in Chrome"** to open the file directly in Chrome.
4. Alternatively, select **"Live Server (Chrome)"** if you have a local server running on port 3000.

### Option 2: Command Line (Node.js required)
1. Open your terminal.
2. Run the following command to start a local server:
   ```bash
   npm run start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Option 3: Direct Open
Simply double-click `index.html` to open it in your default web browser.
